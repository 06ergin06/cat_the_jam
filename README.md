# King of The 42

Bu proje 42 İstanbul kampüsünde düzenlenen *Cat the jam* etkinliği için **odindar** ve **iergin** tarafından Godot ile geliştirilmiştir. 

### Çalıştırma
#### 1. Yöntem 
Projeyi indirip project.godot dosyasını Godot'ta import edin. Sonrasında "Run in browser" seçeneği ile çalıştırın.
#### 2. Yöntem
export/ klasörünü indirin. Sonrasında aşağıdaki komutla bir localhost başlatın.
```bash
python3 -m http.server 8060
```
``http://localhost:8060/tmp_js_export.html``
 linkiyle oyuna girin.
